## 基本原則

**キーワード列挙ではなく、シーンを説明する。**

```
❌ 猫, かわいい, 窓, 日光
✅ 窓辺で日向ぼっこする白い猫。午後の柔らかい光が毛並みを照らしている
```

## 1. フォトリアリスティック

カメラ用語と照明の詳細を含める。

### 要素

- **カメラ**: 焦点距離、絞り、被写界深度
- **照明**: 自然光、ゴールデンアワー、スタジオ照明
- **構図**: クローズアップ、俯瞰、三分割法

### 例

```
ポートレート写真。85mm f/1.4で撮影。ゴールデンアワーの自然光。
被写体は30代女性、浅い被写界深度で背景ボケ。
三分割法に従った構図。
```

```
建築写真。広角レンズ24mm。青い空を背景にした現代的なガラスビル。
対称的な構図、日中の強いコントラスト。
```

## 2. イラスト・ステッカー

スタイルを明確に指定する。

### スタイル例

| スタイル | 説明 |
|----------|------|
| アニメ風 | 日本のアニメーション風 |
| 水彩画 | 柔らかい色彩、にじみ |
| フラットデザイン | シンプル、影なし |
| ベクターイラスト | 明確な線、幾何学的 |
| ピクセルアート | ドット絵、レトロゲーム風 |

### 例

```
フラットデザインのステッカー。笑顔のコーヒーカップ。
パステルカラー、太い黒のアウトライン、白い背景。
```

```
水彩画風の森の風景。柔らかい緑と青の色彩。
紙のテクスチャ、色のにじみを活かした表現。
```

## 3. テキスト含有画像

フォントと配置を詳細に指定。**Pro モデル推奨**。

### 要素

- **フォント**: セリフ/サンセリフ、太さ、スタイル
- **配置**: 中央、左上、下部など
- **サイズ**: 大きく目立つ、小さく控えめ

### 例

```
「HAPPY BIRTHDAY」と書かれたバースデーカード。
太いサンセリフフォント、中央配置。
背景は風船とケーキのイラスト、パステルピンク基調。
```

```
カフェの看板。「OPEN」の文字、ヴィンテージ風セリフフォント。
木目調の背景、チョークアート風の装飾。
```

## 4. 商品写真

スタジオライティングとカメラアングルを詳述。

### 要素

- **背景**: 白背景、グラデーション、ライフスタイル
- **ライティング**: ソフトボックス、リムライト、リフレクター
- **アングル**: 正面、45度、俯瞰、クローズアップ

### 例

```
白い背景の商品写真。ワイヤレスイヤホン。
ソフトボックスで均一なライティング。
45度アングル、影を最小限に。4K解像度。
```

```
コスメ商品のライフスタイル写真。リップスティック。
大理石のテーブル、自然光、花と一緒に配置。
正面やや上からのアングル。
```

## 5. ミニマルデザイン

ネガティブスペースを活用。背景画像に最適。

### 要素

- **余白**: 広い空間を意識的に残す
- **色数**: 2-3色に限定
- **要素**: 必要最小限のオブジェクト

### 例

```
ミニマルな抽象背景。パステルブルーのグラデーション。
画面の右下に小さな幾何学的形状。
大きな余白、クリーンでモダンな印象。
```

```
シンプルなデスクトップ壁紙。白い背景に一輪の花。
中央やや下に配置、周囲に十分な余白。
```

## Google検索グラウンディング活用例

`--search` オプションで最新情報を反映した画像を生成。

```bash
# 2024年の最新ファッショントレンドを反映
python scripts/generate_image.py "2024年秋のファッショントレンドを着た女性" --search -o fashion.png

# 最新のスマートフォンデザインを参考に
python scripts/generate_image.py "最新のスマートフォンのコンセプトデザイン" --search -o phone.png
```

## 編集時のプロンプト

### 追加

```
「〇〇を追加して」「〇〇を入れて」
例: 「背景に虹を追加して」
```

### 削除

```
「〇〇を削除して」「〇〇を消して」
例: 「背景の人物を削除して」
```

### 変更

```
「〇〇を△△に変更して」「〇〇を△△にして」
例: 「髪の色を金髪に変更して」
```

### スタイル変換

```
「〇〇風にして」「〇〇スタイルに変換して」
例: 「アニメ風にして」「油絵風に変換して」
```
